## Support

Thanks for using Premix ORM. Please use the channels below in order.

1) Read the docs
- Start with `README.md` and `docs/`.
- For CLI details, see `orm-book/`.

2) Search existing issues
- Check for duplicates before opening a new one.

3) Open a new issue
- Use a clear title and a minimal reproduction.
- Include OS, Rust version, and relevant logs.

Security issues
- Please follow `SECURITY.md` for responsible disclosure.

Scope
- We provide best-effort help for supported use cases and recent versions.
